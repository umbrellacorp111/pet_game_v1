# ⚔️ Питомец Канала v5 — PvP-АРЕНА

Главная петля: **играешь и ухаживаешь → копится ЗАРЯД АРЕНЫ → при 100%
бой с другим подписчиком → XP, трофеи, лиги и ЖЕТОНЫ 🎟 → эксклюзивные
украшения питомца, которые за монеты не купить.**

## Как работает Арена
1. **Заряд**: раунд мини-игры +34%, кормление/мытьё/сон +10%. Новичок
   стартует с 60% — до первого боя рукой подать.
2. **Подбор**: сервер находит соперника с ближайшими трофеями (призрак —
   его недавний результат). Если игроков мало — тренировочный бот.
3. **VS-экран**: питомцы, уровни, лиги — и твой бонус ухода.
4. **Бой**: раунд «Лови еду», очки × бонус ухода (до +15% при полных
   шкалах). Уход теперь напрямую влияет на победу.
5. **Итог**: победа +20 🏆 +3 🎟 +40 XP · поражение −10 🏆 +1 🎟 +12 XP.
   Заряд обнуляется — цикл начинается заново.

## Лиги и сезоны
🪵 Деревяшка → 🥉 Бронза (100) → 🥈 Серебро (250) → 🥇 Золото (500) →
💎 Алмаз (900) → 🔥 Легенда (1500). Каждую неделю сезон закрывается:
жетоны по лиге (2…50 🎟), трофеи делятся пополам. Топ канала теперь
сортируется по трофеям — это рейтинг бойцов.

## Витрина Арены (только за 🎟)
Шлем викинга 10 · Пылающий взгляд 18 · Шляпа мага 25 · Колизей 30 ·
Аура грозы 40 · Корона чемпиона 60. Статусные вещи: видно, что владелец
добыл их в боях.

## Честность (античит)
Результат соперника сервер знает заранее; твой раунд проверяется по
токену/времени/темпу; бонус ухода считает сервер. Ставок монетами на
исход нет — награды только от системы (важно для аудитории 13–17).

## Файлы и запуск — как в v4
bot.py + static/{index.html, style.css, app.js, pet3d.js, games.js}.
BotHost: переменные BOT_TOKEN, WEBAPP_URL (https!), CHANNEL_ID, PORT;
команда `python bot.py`.

## Тюнинг арены (константы в bot.py)
CHARGE_PER_GAME/CARE, ARENA_WIN/LOSE_*, CARE_BONUS_MAX, LEAGUES, ARENA_SHOP.

### Что реализовано
- **Первый запуск**: чёрный экран → звёзды → портал → титул → живая 3D-сцена
  выбора героя (он/она дышат, моргают, смотрят на игрока; тап — реакция и
  улыбка) → портальный переход в комнату. Экранов загрузки нет.
- **Живой герой**: скелет из пивот-костей (голова/позвоночник/плечи/локти/
  бёдра/колени/хвост волос), модульное лицо (зрачки со слежением и микро-
  саккадами, веки, брови, рот с «немой речью», румянец), выбор пола хранится
  в Telegram CloudStorage.
- **Эмоции**: стейт-машина из 10 состояний с интенсивностью; ведёт лицо,
  позу, тинт света, зум камеры, партиклы и свечение UI; базовая эмоция
  синхронизируется с серверными шкалами (голод/усталость и т.д.).
- **Тач-зоны**: голова/лицо/тело/руки/ноги, эскалация при спаме (4 стадии),
  цепные реакции, «голос» героя (невербальные чирпы), микро-тряска камеры.
- **Idle/AFK-автономия**: взвешенные микро-действия без повторов; 30с —
  оглядывается, 60с — садится, 150с — отдыхает; серверный сон = поза сна,
  Zzz-партиклы, пробуждение с потягиванием.
- **Камера**: демпфирование, дыхание, параллакс за пальцем, импульсы зума,
  тряска на событиях, эмо-смещение.
- **Свет и миры**: свет per-комната, золотая вспышка на легендарку,
  экипированные фоны меняют мир (закат/космос/неон/колизей).
- **Партиклы**: пул без аллокаций в кадре, амбиент комнат (пыль/неон/пузыри/
  искры), деградация бюджета при падении FPS.
- **UI как часть мира**: стеклянный HUD со свечением при изменении значений,
  плавающее орб-меню с радиальным раскрытием, редкость предметов с VFX
  покупки по тирам (легендарная: вспышка света + тряска + шторм партиклов),
  всплывающие уведомления, событие дня.
- **Арена**: VS-экран, отсчёт 3-2-1-БОЙ, тот же честный движок «Лови еду»
  (комбо — только визуал), победный танец/поражение героя, анимированные
  дельты наград. Мини-игры и античит-контракты не тронуты.

Деплой прежний: `python bot.py`, переменные BOT_TOKEN, WEBAPP_URL, CHANNEL_ID, PORT.
